VISUAL TECHNIQUES — USE ALL OF THESE:

1. NOISE TEXTURE OVERLAY
Add a subtle grain/noise texture over the hero using this CSS:
.noise::after {
  content: '';
  position: absolute;
  inset: 0;
  background-image: url("data:image/svg+xml,%3Csvg viewBox='0 0 256 256' xmlns='http://www.w3.org/2000/svg'%3E%3Cfilter id='noise'%3E%3CfeTurbulence type='fractalNoise' baseFrequency='0.9' numOctaves='4' stitchTiles='stitch'/%3E%3C/filter%3E%3Crect width='100%25' height='100%25' filter='url(%23noise)' opacity='0.04'/%3E%3C/svg%3E");
  pointer-events: none;
  z-index: 1;
}

2. GLASSMORPHISM CARDS
Cards use: background: rgba(255,255,255,0.03); backdrop-filter: blur(20px); border: 1px solid rgba(255,255,255,0.06);
For light themes: background: rgba(255,255,255,0.7); backdrop-filter: blur(20px); border: 1px solid rgba(0,0,0,0.06);

3. GRADIENT TEXT HEADINGS
Hero business name uses gradient text:
background: linear-gradient(135deg, [accent-light] 0%, [accent] 50%, [text-primary] 100%);
-webkit-background-clip: text;
-webkit-text-fill-color: transparent;
background-clip: text;

4. ANIMATED GRADIENT BACKGROUND
Hero background uses a slow-moving gradient animation:
@keyframes gradientShift {
  0% { background-position: 0% 50%; }
  50% { background-position: 100% 50%; }
  100% { background-position: 0% 50%; }
}
background-size: 400% 400%;
animation: gradientShift 12s ease infinite;

5. SCROLL REVEAL ANIMATIONS
Elements animate in as they enter the viewport using Intersection Observer:
- Fade up: opacity 0→1, translateY 30px→0, duration 0.6s
- Stagger children with 0.1s delay increments
- Use a useInView hook pattern

6. BENTO GRID LAYOUT for services section
Instead of a boring 3-col grid, use an asymmetric bento layout:
- First card spans 2 columns and is taller (featured)
- Remaining cards fill in a mosaic pattern
- Cards have different heights for visual rhythm

7. MAGNETIC HOVER EFFECT on CTA buttons
Buttons subtly follow the cursor on hover:
onMouseMove: calculate cursor offset from button center, apply small transform
Max offset: 8px x, 4px y

8. MARQUEE/TICKER for social proof or business type keywords
A horizontal scrolling ticker between hero and services:
- Shows business keywords, awards, or "Premium · Trusted · Loved by locals ·" etc.
- Infinite loop, smooth scroll
- Use CSS animation, no JS library

9. FULL-BLEED SECTION DIVIDERS
Between sections, use one of:
- SVG wave divider (inline, matching next section background)
- Diagonal clip-path: clip-path: polygon(0 0, 100% 0, 100% 85%, 0 100%)
- Thick accent color border: border-top: 3px solid [accent]

10. FLOATING BADGE / SOCIAL PROOF ELEMENT
Floating card in hero (bottom-left or bottom-right, absolute positioned):
- Shows: ⭐ 5.0 · "Trusted by 200+ clients" or similar
- Glassmorphism style
- Subtle bounce animation
